Install Snifferdog

Firefox
Click here.

Chrome
Paste this into the address bar. Make sure the 'javascript:' is there! Optionally change the keyword to 's' or 'sni'.
javascript:window.external.AddSearchProvider("http://127.0.0.1:43110/19X5gH4DnWFGhjYCxHzun9BYmCJvBsHjgc/opensearch.xml");

Opera
Right click on the address bar and select 'Edit Search Engines...'. Click 'add new search'. Enter the following information:
Name: Snifferdog
Keyword: s
Address: http://127.0.0.1:43110/19X5gH4DnWFGhjYCxHzun9BYmCJvBsHjgc/?q=%s
Use Post: No/unchecked.
Query String: Blank/Greyed out.
Safari
At the moment Safari does not support external search engines.
